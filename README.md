## Setting up

1. Clone the repo: `$ git clone [URL]`

2. Install dependencies: `$ bundle`

3. Run the app: `$ ruby app.rb`

4. Navigate to [http://localhost:4567/](http://localhost:4567/)

Use `$ rspec` to run tests
